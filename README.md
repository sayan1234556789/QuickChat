🚀 QuickChat – Real-Time Chat Application

🔗 GitHub Repository:
https://github.com/sayan1234556789/QuickChat

📌 Overview

QuickChat is a full-stack real-time chat application that enables users to communicate instantly through text messages, group chats, and video calls. Built with modern web technologies, it ensures seamless communication with low latency and secure authentication.

✨ Features

💬 Real-Time Messaging
Instant one-to-one communication using Socket.io

👥 Group Chat Functionality (NEW 🚀)
Create and participate in group conversations in real-time

🟢 Live User Interaction
See active users and chat instantly

🎥 One-to-One Video Calling
Peer-to-peer video calling using WebRTC

🔐 Secure Authentication & Authorization
Protected routes and user sessions

⚡ Fast & Scalable Backend
Efficient handling of real-time events

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/sayan1234556789/QuickChat.git
cd QuickChat
2️⃣ Install dependencies

Backend:

cd server
npm install

Frontend:

cd client
npm install
3️⃣ Environment Variables

Create a .env file in the server folder:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
4️⃣ Run the application

Start Backend:

cd server
npm run dev

Start Frontend:

cd client
npm start
